# Business Rules

This document outlines the core logic and rules governing the Connect Digital School platform.

## 1. User Roles

### Admin/Owner
- Full platform control.
- Configure payment models.
- Manage content (courses, modules, lessons).
- View analytics and reports.
- Manage users and permissions.

### Teacher
- Create and edit lessons.
- Upload videos.
- Create quizzes and assessments.
- Grade student work.
- Moderate forum.
- View student progress.

### Student
- Access purchased content.
- Watch video lessons.
- Take quizzes and exams.
- Participate in forums.
- Track progress and gamification.
- Receive certificates.

## 2. Payment Models (Owner Configurable)
- **Per Module:** One-time payment per course module.
- **Monthly Subscription:** Recurring monthly access.
- **Annual Subscription:** Recurring yearly access (with discount).
- **Weekly Subscription:** Recurring weekly access.
- **Lifetime Access:** One-time payment for permanent access.

## 3. Level Assessment System
- **Placement Test:** New users take a 20-question test to determine their initial level.
- **Level Classification:** A1, A2, B1, B2, C1, C2 (CEFR framework).
- **Progress Exams:** Students must pass a final exam at the end of each level.
- **Minimum Score:** 70% required to advance to the next level.
- **Retake Policy:** 3 attempts per exam with a 24h cooldown between attempts.

## 4. Gamification System
- **Points:** Earned by completing lessons (50 pts), quizzes (based on score), and forum participation.
- **Badges:** 30+ unique achievements (e.g., "First Lesson", "7-Day Streak").
- **Levels:** Student rank progression (Bronze → Silver → Gold → Platinum → Diamond).
- **Streaks:** Daily login and activity tracking.
- **Leaderboards:** Global and class-based rankings.

## 5. Certificate Generation
- **Criteria:** 100% module completion + Final Exam score ≥ 80%.
- **Verification:** Every certificate includes a unique ID and a QR code for public verification.
- **Delivery:** Sent automatically via Resend and available for download in the profile.
